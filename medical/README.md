## 📹 How to Add Your Clinic Video:

### STEP 1: Create/Record Your Video
Choose one method:
- **OBS Studio** (FREE) - Professional screen/camera recording
  - Download: https://obsproject.com
  - Record 2-3 minute clinic tour
  - Export as MP4
  
- **Your Smartphone** - Use iPhone/Android camera
  - Record clinic footage
  - Transfer to computer
  
- **DaVinci Resolve** (FREE) - Video editing
  - Download: https://www.blackmagicdesign.com/products/davinciresolve
  - Add music, transitions, text overlays
  - Include voiceover narration

### STEP 2: Video Content Should Include:
- Reception area
- Waiting room
- Consultation rooms
- Medical equipment
- Professional background music
- Clear audio (voiceover optional)
- Text overlays with clinic name

### STEP 3: Save Your Video
1. Save as: **clinic_tour.mp4** (MP4 format only)
2. Place in: **assets/videos/** folder
3. That's it! The website will automatically display it

## 🆘 Troubleshooting:

**Video not showing?**
- Ensure file is named: clinic_tour.mp4
- Check location: assets/videos/clinic_tour.mp4
- Verify MP4 format compatibility

**Styling looks broken?**
- Check CSS file is in css/style.css
- Clear browser cache (Ctrl+Shift+Delete)

**Forms not working?**
- Use Formspree or similar service
- Connect to Google Sheets
- Use PHP backend
